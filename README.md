# Approximately Up Mods site

Static GitHub Pages catalog for Approximately Up mods.

## Files

- `index.html` - main catalog page.
- `upload.html` - mod submission mockup and live preview page.
- `script.js` - catalog, local profile, language switching and mod details.
- `upload.js` - upload form preview and local-only submit notice.
- `styles.css` - styling.
- `assets/` - screenshots, backgrounds and avatars.
- `downloads/` - downloadable mod archives.

## Editing

Open the folder in VS Code or any editor, change the HTML/CSS/JS files, then preview `index.html` in a browser.

## Publishing

Repository: https://github.com/cucumber666777/approximately-up-mods
Pages URL: https://cucumber666777.github.io/approximately-up-mods/

Use:

```bash
git add .
git commit -m "Update site"
git push origin main
```

## Notes

This package does not include a backend connection or private database keys. The upload page is a frontend form/preview only until a backend is added again.
