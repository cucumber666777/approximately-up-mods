Minimum rules

  • Upload only `.zip`, `.dll`, or a ZIP package that contains the mod and README files.
  • Use clear screenshots when the mod changes gameplay, visuals, planets, tools, menus, or blocks.
  • Write what the mod does, what game build it supports, and how to install it.
  • Do not upload harmful files, stolen work, fake downloads, or misleading descriptions.
  • Submitted mods are hidden until the site owner manually approves them.
